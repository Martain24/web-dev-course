Introducción a la CSS box model

CSS considera que todos los elementos HTML están metidos en una caja. La caja tiene metido el contenido del elemento, por ejemplo, si tenemos un párrafo dentro de la caja tendremos los que hayamos escrito en dicho parráfo. Alrededor del contenido tenemos el padding que es el espacio entre el contenido y los bordes de la caja. El background de la caja sería el espacio ocupado por el contenido y el padding.

Entonces después del padding tenemos el border que recuerda que lo podemos cambiar con border radius para que sea redondo. También podemos cambiar el border en sí, podemos cambiar el color del border, su grosor, si se hace con una línea discontinua o no, etc.

Después, alrededor del border, tenemos el margin. El margin es el espacio que queda entre la caja del elemento y los demás elementos que tiene alrededor.

Veamos un ejemplo:
p {
padding: 2px 5px 2px 5px;
border:2px dashed black;
margin: 8px 6px;
marging:8px
}
Con esto estaríamos marcando que nuestro párrafo tiene que tener 2px de padding arriba, 5px a la derecha, 2px abajo y 5 px a la izquierda. Fíjate que es como las agujas del reloj y no hay espacios.

El borde sería 2px de ancho, sería dashed y negro. tienen que ir en este orden y sin comas

Si solo ponemos 2 datos como en margin estaríamos marcado espacio arriba y abajo en ese orden

Si simplemente ponemos margin: 8px se aplicarían esos 8px de margin sobre todos los lados.

Usando Box model

Lo de margin auto te sive para que de forma autómatica el elemento se le ponga un margin de tal manera que quede centrado en el elemento en el que se encuentra metido por ejemplo, si tienes una lista metida en el body y pones margin auto en derecha e izquierda se pondrá en el centrada. Es útil cuando a la lista le defines un width diferente del que viene por defecto. Si quieres tener una lista con un widht más pequeño necesitaras lo de auto para que se centre.

Selectors y combinators

Ojo porque hay cosas importantes. Imaginate que dentro de un elemento A tienes otro elemento B y quieres que el elemento B se vea diferente solo cuando esta dentro del elemento A. Lo que tienes que hacer en CSS es poner:
A B {}
Así estás cambiando el estilo de los elementos B que están dentro de A.

Entiendo las clases

Puedes crear una clase y después aplicársela a cualquier objeto. Es como un id, la diferencia es que el id solo se puede aplicar a un elemento y la clase se puede aplicar a muchos elementos independientemente del elemento que se trate. Realmente puedes hacer magia con esto de las clases y manejarlas como te de la real gana.

Block vs Inline elements

Ojo, porque si poner un párrafo y despues otro párrafo se ponen uno debajo del otro porque son block elements que ocupan todo el widht de la página web. Sin embargo, si pones un anchor tag ("a") y después otro se ponen uno al lado del otro porque son inline elements que no ocupan todo el espacio de la web.

Algunos inline elements comunes: a; button; img; span;

El span es para marcar una parte de un texto en concreto y poder cambiar su estilo.
