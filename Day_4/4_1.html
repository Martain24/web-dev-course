Vídeo 37

Cuando linkeamos un archivo de CSS a un HTML no significa que todo lo de estilos se esté copiando en el HTML. Lo que ocurre es que el navegador primero pide la información al html y después pide la información al archivo de CSS que tenemos vinculado. Lo que va haciendo el navegador es ir pidiendo las diferentes informaciones a los diferentes archivos que estemos usando para crear la página web.

Vídeo 38

A veces queremos aplicar a diferentes objetos de la misma índole (por ejemplo dos párrafos) diferentes estilos. En CSS, mediante un selector de elemento de html accedemos a todos los párrafos a la vez y los cambia todos. Nuestro objetivo es cambiar el estilo para un párrafo en específico, ¿cómo podemos hacerlo? podriamos aplica inline CSS, pero no queremos juntar CSS y html en un mismo archivo. Usaremos el atributo "id" para crear un id para un elemento en concreto y crearemos un style específico en nuestro archivo de css. La clave es que en CSS simplemente pondremos "#idname" para cambiar el estilo. Algo importante es que el id tiene que ser único para cada elemento, no debemos usar el id en dos elementos a la vez.
