El elemento de style de HTML no es visible en la página web. Es un elemento que añade Metadata a nuestra página para que los elementos visibles se vean de formas diferentes. Es información importante para el navegador, pero no para el usuario.

Anatomía de un HTML válido

Todos los elementos visibles para el usuario deben estar métidos dentro de un elemento que llamaremos body. En el elemento head pondremos Metada de la página que no es visible para el usario como por ejemplo el elemento de style o de título. También debemos poner tanto el head como el body envueltos en el elemento html y arriba de todo debemos poner lo de DOCTYPE html.
