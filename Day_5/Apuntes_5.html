Añadir imágenes

Ya sabemos añadir imágenes a nuestra página web:



Aquí tenemos el código para poner una imagen. Fíjate que simplemente usamos "img" y en "src" ponemos los que aparecería si el navegador no es compatible con dicha imagen. Para centrarla necesitamos centrar el elemento que este dentro de la imagen no podemos poner como atributo de CSS en img lo de text aling, porque ese atributo solo afecta a lo que esté dentro del elemento en sí.

Background y añadir segundo HTML

Hemos aprendido a cambiar el color de background de body y también sabemos como conectar dos archivos de HTML mediante un link, lo cual está de puta madre para hacer cada vez páginas webs más complejas e interesantes.

Aplicar diferentes CSS

Si aplicas diferentes documentos de CSS a un HTML acuerdate de poner primero al documento de shared que se aplica a todos los html y despues escribir el específico. Si el específico hace cambios sobre el shared se produciran los cambios que buscar al haberlo escrito después.
