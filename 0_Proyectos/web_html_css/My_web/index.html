HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses elements to describe (annotate) content
  • HTML elements tipically have an opening tag, content and then a closing tag
  • You can also void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you'll gain experience over time, no worries

Learn more about all availablre HTML elements on the MDN element reference

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content

  • Styles are assigned via property-value pairs
  • You can assing styles via the "style" attribute
  • To avoid code duplication, you tipically use global styles (e.g. via "style" element) instead
  • Alternatively, you can work with external stylesheets files which you "link" to
  • When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all availablre CSS properties on the MDN property reference
